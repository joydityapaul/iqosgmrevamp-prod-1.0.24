# GM-REVAMP 
 
## Clone the repository from PSCode Lion Cloud to your local machine using the following command:

    git clone https://pscode.lioncloud.net/jassingh39/iqos-gm-revamp-dev.git
## Open the cloned project in Visual Studio Code.Verify that all dependencies required for the project are installed. You can do this by running the following command in the Visual Studio Code terminal from the project directory:

    npm install

## Start the application by running the following command from the root directory of the project.

    node app.js

## This will start the server at http://localhost:5000
## Open your preferred web browser and navigate to http://localhost:5000/.
## You should see a welcome message indicating that the server is running correctly.
